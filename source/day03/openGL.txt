openGL:光照
1.环境光,ambient,没有方向.
2.散射光:diffuse,具有方向性,根据入射光角度在表面均匀反射开来.
3.镜面光:specular,具有很强的方向性.沿特定的角度进行反射.

光源由三种光成分构成.
使用光照时,物体具有材料的特征.

material:材料
1.一般将材料的环境光和散射光的反射率设置为相同的值.
2.颜色追踪:跟踪,通过调用glColor函数来设置材料的反射率.
		gl.glEnable(gl_color_material);
		gl.glColorMaterial(gl_front,gl_ambient_and_diffuse);

openGL至少支持8种独立的光源.
法线:垂线.
向量:表示方向的量.也叫方向量.

法线扭曲模拟曲面.
单位法线:长度为1的法线.
法线规范化:计算法线长度,达到1.

镜面指数:指定镜面加亮的大小和集中性.范围1-128.

spot_cutoff:切角.

混合效果:开启后,深度测试失效.

Blend:混合.

//
glBlendFunc();//设置混合因子
目标颜色:存储在颜色缓冲区中的颜色值.
源颜色:即将进入颜色缓冲区的值.
混合方程式:
S:源颜色的混合因子.
D:目标颜色的混合因子.

Cd:1,0,0,0,
Cs:0,0,1,0.5

S:0.5
D:1 - 0.5 = 0.5
Cf = Cs x S + Cd x D.

rgb = (0*0.5 + 1 * 0.5 )  ()  ()

抗锯齿:反锯齿.反走样

fog:雾.
Dither:抖动.

纹理:texture, map
每个纹理贴图绑定一个整型值上,纹理id.
GL_TEXTURE_MAG_FILTER:放大过滤器
